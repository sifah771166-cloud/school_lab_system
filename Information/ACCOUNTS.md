## 🔑 Cara Login

1. Buka aplikasi di browser
2. Masukkan email dan password sesuai role yang ingin ditest
3. Klik tombol "Login"

### Contoh Login Super Admin:
```
Email: admin@school.com
Password: superadmin123
```

### Contoh Login Admin TKJ:
```
Email: admin.tkj@school.com
Password: admintkj123
```

### Contoh Login User DKV:
```

Password: userdkv1
```
